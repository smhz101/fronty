== Description ==

Fronty is a versatile, performance-focused block theme built for the modern web. It fully supports Full Site Editing (FSE), allowing you to customize every aspect of your site directly from the WordPress editor. With over 50+ block patterns, 7 distinct style variations, and a comprehensive design system, Fronty makes it easy to create professional websites for any nicheâ€”from gyms and restaurants to portfolios and SaaS landing pages.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file.
3. Click Install Now.
4. Click Activate to use your new theme right away.
