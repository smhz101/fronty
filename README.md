# Fronty - Modern WordPress Block Theme

Fronty is a powerful, versatile, and modern WordPress block theme designed for Full Site Editing (FSE). It features a comprehensive design system, fluid typography, and over 50+ patterns to help you build any type of website.

## Features

-   **Full Site Editing (FSE)**: Customize every part of your site using the Block Editor.
-   **50+ Block Patterns**: Ready-to-use patterns for heroes, features, testimonials, pricing, and more.
-   **7 Style Variations**: Instantly switch between Default, Dark, Vibrant, Minimal, Nature, Warm, and Ocean styles.
-   **Fluid Typography & Spacing**: Responsive design that scales perfectly across all devices.
-   **Niche Homepages**: Pre-built layouts for Gyms, Beauty Salons, Restaurants, Lawyers, and more.
-   **Advanced Layouts**: Sticky scroll sections, full-screen video heroes, and funnel pages.
-   **Performance Optimized**: Zero JavaScript dependencies for the frontend, using only native WordPress block styles.

## Installation

1.  Download the `fronty.zip` file.
2.  Go to **Appearance > Themes** in your WordPress dashboard.
3.  Click **Add New** and then **Upload Theme**.
4.  Upload `fronty.zip` and click **Install Now**.
5.  Activate the theme.

## Getting Started

1.  **Explore Patterns**: Go to the Block Editor and open the pattern inserter to see all available patterns.
2.  **Choose a Style**: Go to **Editor > Styles** to browse and select a style variation.
3.  **Customize**: Use the Global Styles interface to tweak colors, fonts, and layout settings.
